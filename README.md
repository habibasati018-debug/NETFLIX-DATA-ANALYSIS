# 🎬 NETFLIX Movies & TV Shows Data Analysis Dashboard

## 📌 Project Overview
This project analyzes Movies and TV Shows data using Python for Exploratory Data Analysis (EDA) and visualization techniques to uncover patterns in content distribution, ratings, and country-wise production trends.

The goal is to understand how content varies by type, rating, and geography.

## 🔍 Exploratory Data Analysis (EDA)

The following analyses were performed:

- 📊 Movies vs TV Shows distribution (Bar Chart)
- ⭐ Rating distribution across content (Bar Chart)
- 🔗 Relationship between type and ratings
- 🌍 Country-wise content distribution (Word Cloud)

---

## 📈 Key Findings

- Movies dominate the dataset compared to TV Shows.
- Most content falls under common ratings such as **TV-MA** and **TV-14**.
- TV Shows tend to have more mature ratings compared to Movies.
- Certain countries contribute significantly more content than others.
